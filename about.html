About the Predictive Maintenance Web App

This web application helps manufacturing industries monitor and predict machine failures using a machine learning model.

Users can input real-time machine data such as temperature, torque, and rotational speed to determine the likelihood of machine failure.

The model behind this app is trained using Random Forest Classifier and leverages historical maintenance data to provide accurate predictions.

Predictions are saved in a MySQL database and can be reviewed anytime using the History tab.

Designed for smart factories to reduce downtime, improve maintenance planning, and save costs.
